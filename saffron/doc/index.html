Saffron

Saffron is an extension to Java to incorporate SQL-like relational expressions. It features a pre-processor, a query optimizer, and a runtime library.

It is also a component, with sister projects farrago and fennel, of an open-source RDBMS.

Read the overview, installation instructions, and javadoc.

The project is hosted at SourceForge. The source code (including this page) is hosted in the Eigenbase perforce server. Thanks to Eigenbase, and to Perforce for letting us use their server gratis.
