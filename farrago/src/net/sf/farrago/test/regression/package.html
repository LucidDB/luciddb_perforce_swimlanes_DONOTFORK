Farrago Regression Testing

Whereas the Farrago unit test framework is focused on low-level testing of specific components, regression tests provide more exhaustive coverage at the feature level. However, the two frameworks share quite a bit of infrastructure. Regression tests may be either SQL-based or Java-based; scripts for SQL-based tests should be created with their reference logs under the //open/dev/farrago/regressionsql directory. {@link net.sf.farrago.test.regression.FarragoSqlRegressionTest} takes care of running these SQL scripts multiple times with various combinations of system parameter settings for coverage.
