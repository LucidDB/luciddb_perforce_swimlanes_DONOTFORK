Defines Farrago-specific datatype support.
This package subclasses generic datatype interfaces from {@link org.eigenbase.reltype} and {@link org.eigenbase.oj} in order to implement true SQL datatype semantics (e.g. nullability, precision, scale, character set, and collation sequence). During query processing, the Farrago JDBC driver ensures that an instance of {@link net.sf.farrago.type.FarragoTypeFactory} is used for all type analysis.
