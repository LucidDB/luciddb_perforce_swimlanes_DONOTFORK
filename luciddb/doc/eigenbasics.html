LucidDB/Eigenbase Relationship

LucidDB is actually implemented as a set of plugins into frameworks developed by The Eigenbase Project, along with wrapper scripts and other customizations. Eigenbase provides a system-level extensibility framework called Farrago, which also integrates the Fennel framework for native-code portions. For more information on how these frameworks make possible extension projects such as LucidDB, see the Farrago architecture doc.
